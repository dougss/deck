# Bundled fonts — licenses & sources

All `.woff2` files in this directory are bundled with the Deck app
to avoid runtime network dependencies (constitution §5).

## Clash Display

- **Files:** `ClashDisplay-Semibold.woff2`, `ClashDisplay-Bold.woff2`
- **Weights:** 600, 700
- **Source:** Fontshare (Indian Type Foundry)
  `https://api.fontshare.com/v2/fonts/download/clash-display`
- **License:** Fontshare Free Font License
  (https://www.fontshare.com/licenses)
  Permits embedding in commercial desktop applications and
  redistribution as part of the app. Does not permit redistributing
  the font files as a standalone catalog.
- **Downloaded:** 2026-04-21

## Inter

- **Files:** `Inter-Regular.woff2`, `Inter-Medium.woff2`,
  `Inter-SemiBold.woff2`
- **Weights:** 400, 500, 600
- **Source:** Rasmus Andersson, v4.0
  `https://github.com/rsms/inter/releases/tag/v4.0`
- **License:** SIL Open Font License 1.1
  (https://github.com/rsms/inter/blob/master/LICENSE.txt)
- **Downloaded:** 2026-04-21

## Geist Mono

- **Files:** `GeistMono-Regular.woff2`, `GeistMono-SemiBold.woff2`
- **Weights:** 400, 600
- **Source:** Vercel, v1.8.0
  `https://github.com/vercel/geist-font/releases/tag/1.8.0`
- **License:** SIL Open Font License 1.1
  (https://github.com/vercel/geist-font/blob/main/LICENSE.TXT)
- **Note:** v1.8.0 does not ship a standalone `Medium` (500) weight
  — only `MediumItalic`. Substituted 500 → 600 (SemiBold) for
  emphasis contexts (search kbd, badges).
- **Downloaded:** 2026-04-21
