Design invariants · Phase 3 right sidebar

  • Activity Bar: 48px, anchored to window's right edge, always visible.
  • Panel: 420px wide, opens at right: 48px (glued to activity bar's left).
  • Default mode: overlay. Panel floats over main with shadow.
  • Pinned mode: panel pushes main; main shrinks by 420px; no shadow.
  • Pin toggle: icon in panel header. State persisted in Zustand (rightPanelPinned).
  • ⌘\: toggles panel visibility. Remembers last active icon.
  • Active icon indicator: 2px accent bar on the left of the icon (faces the panel).
  • Planner: kind='planner' session, persists across restarts via claude -c.
  • Terminal: global ephemeral shell, cwd frozen at spawn, stays alive until ⌘Q.
